# Rayon User App Public Overview

## English

Rayon is a real product. The full application and the complete operational codebase are maintained privately.

This public repository exists only to make the project visible at a high level and to provide a small set of representative code samples. It is meant to give readers a general understanding of the product direction, interface style, and development approach without exposing the actual private codebase.

This repository is not the full application.

It is a limited public-facing sample prepared for general information purposes.

### What this repository is for

- making the existence of the project publicly visible
- presenting the general direction of the user application
- sharing a few safe and representative Flutter UI samples
- giving a high-level idea of the product without revealing private implementation details

### What is included here

- selected Flutter sample screens
- mock data used by the sample files
- a simplified repository structure that is safe to review publicly

### What is intentionally not included

- the full production source code
- backend systems and operational integrations
- Firebase-connected runtime flows and private project wiring
- Android and iOS project scaffolding from the original app
- private business logic, internal services, and infrastructure details
- credentials, secrets, service accounts, or production configuration

### Included sample files

- `samples/flutter/home_showcase_sample.dart`
- `samples/flutter/restaurant_menu_showcase_sample.dart`
- `samples/flutter/order_history_showcase_sample.dart`
- `samples/flutter/mock_data.dart`

### Note

Anyone reading this repository should understand the following:

- the project is real
- the actual product is private
- this repository is only a general public overview with limited sample code
